# 🖼️ PNG Converter

A full-stack web application that converts and compresses images into PNG format using advanced algorithms like Huffman Coding and Run-Length Encoding (RLE).

---

## 🚀 Features

- 📤 Upload images via drag & drop or file picker
- 🔄 Convert images to PNG format
- 🧠 Multiple compression algorithms:
  - Simple PNG (Deflate)
  - Huffman Coding
  - Run-Length Encoding (RLE)
- 📊 Displays:
  - Image dimensions
  - Original vs converted size
  - Compression ratio
  - Conversion time
- 👀 Before vs After preview
- 📥 Download converted image

---
